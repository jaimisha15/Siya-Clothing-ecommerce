# Siya Clothing - Luxury Streetwear

A modern, responsive, and premium e-commerce front-end for **Siya Clothing**, a luxury streetwear brand. This project features a clean and elegant user interface, smooth animations, and a seamless shopping experience using vanilla web technologies.

## ✨ Features

- **Responsive Design**: Fully responsive layout optimized for mobile, tablet, and desktop devices.
- **Dynamic Product Catalog**: Data-driven product listing and details pages loaded dynamically via JavaScript.
- **Interactive UI**: Custom flyout menus for the Shopping Cart and Wishlist with interactive state management.
- **Rich Content Pages**: Includes dedicated pages for About, Blog, Contact, Affiliate Program, Video Marketing, and a SWOT analysis.
- **Modern Aesthetics**: Premium styling with fluid typography, CSS variables for theming, and smooth transitions/micro-animations.
- **User Feedback**: Interactive toast notifications when adding items to the cart or wishlist.

## 🛠️ Tech Stack

- **HTML5**: Semantic elements and accessible structure.
- **CSS3**: Custom properties (variables), Flexbox, CSS Grid, custom scrollbars, and responsive media queries.
- **Vanilla JavaScript**: Dynamic DOM manipulation, state management (cart and wishlist), and event handling without external libraries.

## 📂 Project Structure

- `index.html` - The home page featuring hero section, categories, and featured products.
- `products.html` - The main shop page displaying all products.
- `product-detail.html` - Dynamic product details page.
- `about.html` - Brand story and information.
- `contact.html` - Customer support and contact form.
- `blog.html` / `blog-*.html` - Blog listing and individual article pages.
- `swot.html` - Business SWOT analysis.
- `affiliate-program.html` - Details about the brand's affiliate program.
- `video-marketing.html` - Video marketing content.
- `styles.css` - Global stylesheet containing all design tokens and component styles.
- `app.js` - Main JavaScript file handling UI interactions, cart logic, and page routines.
- `product-data.js` - Centralized product catalog data (Mock database).
- `manifest.json` - Web App Manifest.

## 🚀 How to Run

Since this is a front-end project built with vanilla web technologies, no compilation or build step is required.

1. Clone or download the repository to your local machine.
2. Open the `index.html` file directly in any modern web browser.
3. *Recommended*: For the best experience (and to avoid CORS issues if fetching local resources), serve the directory using a local web server. If you use VS Code, the [Live Server](https://marketplace.visualstudio.com/items?itemName=ritwickdey.LiveServer) extension is perfect for this.
